## [2026-05-23] - Маркетинговая кампания: рассылка, deep link, статистика

### Добавлено
- **Phase 1: рассылка v2** в `handlers/admins/broadcast.py`:
  - Поддержка фото с подписью (лимит 1024 симв)
  - Двухступенчатая отправка: "Тест себе" + "Тест на 7882817110" + "Отправить всем"
  - Чтение списка исключений из `data/broadcast_excludes.txt`
  - 6 user_id в исключениях по запросу владельца
- **Phase 2: cross-bot фильтр** — в hamster26 опция "Только не в hamster93 (уникальные)"
- **Phase 3: deep link с автоприменением промокода**:
  - Новые таблицы `broadcasts` и `broadcast_clicks` в БД hamster26
  - В `handlers/start.py` (hamster26): обработка `/start p<broadcast_id>`
  - Дифференцированный UX для активных/неактивных подписчиков
  - Inline-кнопка под рассылкой с URL `t.me/hamster26_bot?start=p<id>`
- **Phase 4: статистика рассылок** в `handlers/admins/stats.py`:
  - Кнопка "📊 Статистика рассылок" в админ-меню обоих ботов
  - По каждой рассылке: доставка, CTR, конверсия в оплату, выручка
  - Pagination по 10 на страницу
- **Phase 5: промокод `INDIVIDUAL2026`** — 33% скидка до 31.05.2026
- **Скрипт выгрузки БД** — объединение hamster93+hamster26 с дедупликацией, экспорт в Excel

### Изменено
- `handlers/start.py` (hamster26) — добавлена ветка для deep link, обычный `/start` без параметра работает идентично оригиналу
- `keyboards/admin.py` — добавлены кнопки "Рассылка" и "📊 Статистика рассылок"
- `handlers/admins/__init__.py` — регистрация новых модулей `broadcast` и `stats`
- `database/keys.py` (hamster26) — две новые таблицы для campaign-данных

### Архитектурные решения
- Все campaign-данные хранятся в **единой БД hamster26**. Hamster93 пишет туда через прямой SQLite-коннект (абсолютный путь).
- Рассылки идут из hamster93, но кнопки ведут в hamster26 — оба бота работают как единая система.
- Промокод применяется только через клик на кнопку, никогда не показывается текстом.

### Известные ограничения
- aiogram использует MemoryStorage. Рестарт бота во время кампании = потеря FSM state у юзеров, кликнувших но не оплативших.
- Цена 1000₽ захардкожена в start.py — работает только с этим промокодом.
