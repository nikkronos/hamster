# PastuhiBot

Telegram-бот для управления подписками и доступа к копиям каналов.

## Описание

PastuhiBot - это Telegram-бот, который:
- Управляет подписками пользователей на доступ к копиям каналов
- Автоматически создает пригласительные ссылки для доступа к каналам
- Управляет доступом пользователей (бан/разбан при истечении подписки)
- Предоставляет админ-панель для управления каналами, подписками и промокодами

## Основные функции

- **Система подписок**: Покупка подписки на 1/3/6/12 месяцев
- **Управление каналами**: Добавление/удаление каналов для отслеживания
- **Пригласительные ссылки**: Автоматическая генерация одноразовых ссылок
- **Админ-панель**: Управление пользователями, каналами, промокодами
- **Планировщик задач**: Автоматические напоминания и управление подписками

## Технологии

- **aiogram 2.25.1** - основной фреймворк для бота
- **Pyrogram 2.0.106** - для юзербота (копирование каналов)
- **APScheduler** - планировщик задач
- **SQLite** - база данных

## Быстрый старт

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` с переменными:
```
TOKEN=your_bot_token
DEVELOPERS=123456789,987654321
ADMINS=123456789
API_ID=your_api_id
API_HASH=your_api_hash
BANK_CARD=your_card_number
```

3. Запустите бота:
```bash
python bot.py
```
